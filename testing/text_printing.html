Printing
toc
Requirements
In an internationalized application, users can successfully print content containing characters from any script.
The most widely used scripts in the world are: Latin, Cyrillic, Simplified Chinese, Traditional Chinese, Korean, Devanagari, Arabic, and the ones used by the Japanese writing system. File operations involving characters from these scripts provides sufficient internationalization testing coverage.
How to test
Sending document content to a printer or PDF
  1. Copy and paste or type the sample strings from each script below to each area of your product that allows sending content to a printer or PDF.
  2. Print the document, and check whether the output matches the sample string.
