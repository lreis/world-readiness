Text Editing
toc
Requirements
In an internationalized application, users can successfully select, delete and navigate through characters from any script.
The most widely used scripts in the world are: Latin, Cyrillic, Simplified Chinese, Traditional Chinese, Korean, Devanagari, Arabic, and the ones used by the Japanese writing system. Text input using characters from these scripts provides sufficient internationalization testing coverage.
Typical usage
Text editing can take place in a variety of places within a software application, for example:
  • Document content
  • Chat windows
  • Comment boxes
Text selection on MacOS / Windows
  1. For each script listed below, copy the sample strings to each editable text area of your product.
  2. Place the cursor to the LEFT of the first character of a string. Then, while pressing the SHIFT key down, press the RIGHT arrow key multiple times. The text selection should expand to the RIGHT with each pressing of the RIGHT arrow key.
  3. Place the cursor to the RIGHT of the last character of a string. Then, while pressing the SHIFT key down, press the LEFT arrow key multiple times. The text selection should expand to the LEFT with each pressing of the LEFT arrow key.
  4. Position the cursor in the middle of a random word, and double-click. The word should be selected.
  5. Position the cursor in the middle of another random word, and triple-click. The paragraph within which the word is located should be selected.

      Text selection on iOS
      1. Type the sample string provided in the previous section in each editable text area of your product. For typing instructions, see the Text Input article.
      2. Double-tap in the middle of the first word, to select the entire word. Two selection bars will be displayed at the beginning and the end of the word. Check that you are able to change the text selection by dragging the bars back and forth.
      3. Tap on the selected text and check that the contextual menu with the Cut, Copy and Paste commands is displayed.
      Text deletion on iOS
      1. Using the sample string from the previous section, select a word or part of a word, and select Cut. Check that the selected text is deleted.
      2. Place the cursor on a different place within the text and press down for a couple of seconds. When the contextual menu pops up, select Paste. Check that the text that was previous cut is properly pasted.
