Text Input
Requirements
In an internationalized application, users can successfully input characters from any script by typing in localized keyboards, or by copying and pasting text from other areas of an application or from other applications, without experiencing character corruption.
The most widely used scripts in the world are: Latin, Cyrillic, Simplified Chinese, Traditional Chinese, Korean, Devanagari, Arabic, and the ones used by the Japanese writing system. Text input using characters from these scripts provides sufficient internationalization testing coverage.
Typical usage
Text input can take place in a variety of places within a software application, for example:
  • Document names
  • Document content
  • Chat windows
  • Comment boxes
Installing and selecting localized keyboards
In order to check text input in multiple scripts, it's necessary to use localized keyboards that contain the necessary international characters. These can be physical keyboards, and/or keyboard layouts provided by the operating system.
Instructions for activating localized keyboard layouts in some operating systems can be found below:
  • Windows
  • MacOS
  • iOS
Text input by typing
Once the proper localized keyboard is selected, proceed to type the sample strings provided for each script, in all areas of the product that you are assessing which accepts text input.
If you are using a keyboard layout with an English (U.S.) physical keyboard, type the keys shown immediately below each sample character.
